Selected Work

Overview

I have been working as a professional front-end developer for seven years. I have built hundreds of websites and consider myself an expert at HTML, CSS, and WordPress.

I also have a new love for JavaScript and have been improving my skills with the language; learning the low-level aspects and using that knowledge to build single page apps in my spare time.

Specialties

I love high-level thinking when it comes to front-end development. I excel at thinking about how to create maintainable and scalable codebases that ensure clarity for all developers.

I was the original author of the concise.css framework and have also used my skills to build an internal front-end framework for the customer-facing aspects of Asana (website, blogs, marketing, internal tools).

I take a component-based approach in developing front-end architectures; breaking CSS into maintainable pieces so that it is easy to understand, maintain, and extend. I have also written many coding style guides to help ensure that code written within the context of a framework is consistently written at a high quality.

If you need somebody to help build a maintainable and scalable user interface, you should get in touch.
