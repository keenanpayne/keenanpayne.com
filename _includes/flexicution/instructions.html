Instructions

This is the prefered method to install Flexicution. Simply install using NPM and include in your Sass file.

{% highlight scss %} // Get Flexicution library npm install flexicution // Import library @import "node_modules/flexicution/flexicution"; {% endhighlight %}

Setup

Flexicution is as agnostic as possible in regards to your layout and pre-defined breakpoints. You can either use the semantic grid mixin to generate rows and columns for your layout and components, or you can generate responsive class names based on the breakpoints used in your project.

Semantic Grid Mixin

{% highlight scss %} // // Generate Semantic Rows // ---------------------- // @param $gutter - Gutter between columns. Default is `$flex-gutter-width` (30px) @include make-row(); // // Generate Semantic Columns // ------------------------- // @param $columns - Number of columns to take up // @param $gutter - Gutter between columns. Default is `$gutter-width` (30px) // @param $media-query - Breakpoint for the column to collapse. Default is null // @param $min-max - Explicitly sets the media query to be `min-width` or `max-width`. Default is `min` @include make-column(); {% endhighlight %}

Generating Class Names

{% highlight scss %} // // Generate Grid Classes // --------------------- // @param $class - Class name you want postfixed after `.col-` // Extra-small @include make-grid(xsmall); // Small @media (min-width: 480px) { @include make-grid(small); } {% endhighlight %}
