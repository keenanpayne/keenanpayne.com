Installation

This is the preferred method to install Flexicution. Simply install using NPM or Bower.

{% highlight bash %} # NPM npm install flexicution # Bower bower install flexicution {% endhighlight %}

Usage

You have two different options for using Flexicution. You may include the framework and build your column classes yourself, or you can use our pre-defined breakpoints and generated class names.

Base

This includes only the Sass mixins that power Flexicution, with no specific class names.

{% highlight scss %} // Import Flexicution library @import "node_modules/flexicution/flexicution"; {% endhighlight %}

With Classes

This includes the Sass mixins that power Flexicution, as well as responsive column classes.

{% highlight scss %} // Import Flexicution library @import "node_modules/flexicution/flexicution-classes"; {% endhighlight %}

Setup

Flexicution is as agnostic as possible in regards to your layout and breakpoints. You can use the semantic grid mixin to generate rows and columns for your layout and components, or you can generate responsive class names based on the breakpoints used in your project.

Semantic Grid Mixin

{% highlight scss %} // // Generate Semantic Rows // ---------------------- // @param $gutter - Gutter between columns. Default is `$flex-gutter-width` (30px) @include make-row(); // // Generate Semantic Columns // ------------------------- // @param $columns - Number of columns to take up // @param $gutter - Gutter between columns. Default is `$gutter-width` (30px) // @param $media-query - Breakpoint for the column to collapse. Default is null // @param $min-max - Explicitly sets the media query to be `min-width` or `max-width`. Default is `min` @include make-column(); {% endhighlight %}

Generating Class Names

{% highlight scss %} // // Generate Grid Classes // --------------------- // @param $class - Class name you want postfixed after `.col-` // Extra-small @include make-grid(xsmall); // Small @media (min-width: 480px) { @include make-grid(small); } {% endhighlight %}

Settings

The Flexicution library has variables that help configure the base library and library with classes. You may override these in your own project if you wish to customize your grid layout.

Base

{% highlight scss %} // Number of grid columns $flex-grid-columns: 12; // Amount of gutter width $flex-grid-gutter-width: 30px; {% endhighlight %}

Classes

{% highlight scss %} // Breakpoints $media-xsmall: 480px; $media-small: 768px; $media-medium: 960px; $media-large: 1120px; $media-xlarge: 1280px; {% endhighlight %}
